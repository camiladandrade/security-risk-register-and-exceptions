## ✅ Definition of Done
Uma entrega está “pronta” quando:
- Está em PT-BR, clara e aplicável
- Traz objetivo, quando usar e outputs
- Define owner/approver quando aplicável
- Está linkada no README raiz e no README da pasta
- Inclui exemplo fictício quando fizer sentido
